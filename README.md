# My Projects

A collection of frontend web experiments, UI components, and mini-projects built with vanilla JavaScript and React.

## Projects

| Folder | Description |
|--------|-------------|
| `todo-app/` | Todo list with local storage persistence |
| `weather-widget/` | Weather widget using OpenWeatherMap API |
| `calculator/` | Scientific calculator with keyboard support |
| `quiz-app/` | Multiple choice quiz with score tracking |
| `countdown-timer/` | Customizable countdown with sound alerts |

## Setup

```bash
git clone https://github.com/NadirAliOfficial/myproject
cd myproject
npm install
npm start
```

Or just open any `index.html` directly in a browser for the vanilla JS projects.

## Tech Stack

- HTML5, CSS3, JavaScript (ES6+)
- React (for component-based projects)
- LocalStorage API
- Public REST APIs (OpenWeatherMap, etc.)
